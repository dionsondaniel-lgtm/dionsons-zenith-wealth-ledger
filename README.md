# Zenith Wealth Ledger

**A Modern, Smart Solution for Personal Finance.** 💰✨

Welcome! This repository hosts the source code for the **Zenith Wealth Ledger**, a specialized Progressive Web App (PWA) designed to help individuals simplify their daily financial operations and debt management.

Unlike generic budgeting software, this was built specifically to solve the headaches of personal finance: tracking multiple income streams, managing complex loan amortization schedules, and keeping on top of due dates.

## 🌟 Why Users Love This
*   **Smart Debt Ledger:** Auto-generates amortization schedules for your loans, whether it's a bank loan or a 0-interest loan from a friend.
*   **Flexible Payments:** Easily record partial payments, custom due dates, and track your progress over time.
*   **Financial Dashboard:** Get a bird's-eye view of your total income, outstanding debt, and upcoming payments with beautiful, interactive charts.
*   **Secure & Private:** Built as a local-first PWA, ensuring your sensitive financial data stays safe and encrypted on your device.

## 🛠️ The Tech Behind It
Built with the latest reliable web technologies:
*   React & TypeScript (Fast & Bug-free)
*   Vite PWA (Offline-ready & Installable)
*   Recharts & Tailwind CSS (Clean, Modern UI)
*   Local Storage (Privacy-focused data management)
